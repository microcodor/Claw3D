# SCR-03 创作工作室布局调整说明

## 📋 调整内容

### 原布局（三行垂直布局）
```
┌─────────────────────────────────┐
│  INPUT · 智能选题                │
├─────────────────────────────────┤
│  PROCESS · 推理过程              │
├─────────────────────────────────┤
│  OUTPUT · AI 生成内容            │
│  (占据剩余空间)                  │
└─────────────────────────────────┘
```

### 新布局（左右两栏布局）
```
┌──────────────────────────┬──────────────┐
│  INPUT · 智能选题         │              │
│  (左侧顶部)              │  PROCESS ·   │
├──────────────────────────┤  推理过程    │
│                          │              │
│  OUTPUT · AI 生成内容     │  (右侧全高)  │
│  (左侧主体，占大部分)     │              │
│                          │              │
└──────────────────────────┴──────────────┘
     左侧 (70%)              右侧 (30%)
```

## 🎯 布局特点

### 左侧栏（70% 宽度）
- **INPUT 区域**（顶部）
  - 显示用户输入的智能选题
  - 固定高度，不占用过多空间
  - 紫色边框高亮（有内容时）

- **OUTPUT 区域**（主体）
  - 显示 AI 生成的内容
  - 占据左侧剩余空间
  - 支持滚动查看长内容
  - 打字机效果输出
  - 绿色边框高亮（生成完成时）

### 右侧栏（30% 宽度）
- **PROCESS 区域**（全高）
  - 显示 4 步推理过程
  - 实时动画效果
  - 步骤完成状态显示
  - 青色边框高亮（思考中）

## 📐 响应式设计

### 宽度比例
- 左侧：`flex: 7` (约 70%)
- 右侧：`flex: 3` (约 30%)

### 间距
- 左右栏间距：`clamp(8px, 1vw, 16px)`
- 左侧 INPUT/OUTPUT 间距：`clamp(6px, 0.8vw, 12px)`

### 最小尺寸保护
- 所有区域都有 `min-width: 0` 防止溢出
- 所有区域都有 `min-height: 0` 确保正确的 flex 行为

## 💡 设计理念

### 为什么这样布局？

1. **内容优先**：生成的内容是最重要的，应该占据最大空间
2. **流程可见**：推理过程放在右侧，用户可以同时看到内容和推理
3. **视觉平衡**：70/30 的比例既突出内容，又保持推理过程的可见性
4. **符合阅读习惯**：从左到右，从输入到输出，再到推理过程

### 与其他屏幕的一致性

- **SCR-01**: 数字办公室（单一视图）
- **SCR-02**: 热搜舆情中心（多区域垂直布局）
- **SCR-03**: 创作工作室（左右两栏布局）✨ 新布局

每个屏幕都有独特的布局，适应其功能特点。

## 📊 布局对比

| 方向 | 垂直（3行） | 水平（2栏） |

| OUTPUT | 底部主体 | 左下主体 |
| 内容可见性 | 较好 | 更好 ✅ |

| 空间利用 | 一般 | 优秀 ✅ |

## 🎉 总结

新布局更加合理，突出了内容生成的重要性，同时保持了推理过程的可见性。左右两栏的设计更符合用户的阅读习惯，提升了整体的用户体验。
