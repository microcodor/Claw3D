# 三屏架构快速启动指南 🚀

## 🎯 快速体验

### 1. 启动开发服务器
```bash
cd Claw3D
npm run dev
```

### 2. 访问页面
打开浏览器访问：
```
http://localhost:3000/office
```

### 3. 操作演示

#### 默认视图（单屏模式）
- 默认显示 **SCR-01 数字办公室**（现有功能）
- 顶部显示三个屏幕标签

#### 切换屏幕
点击顶部标签切换：
- **SCR-01 数字办公室** - 现有的 3D Office 环境
- **SCR-02 热搜舆情中心** - 10个平台实时热搜
- **SCR-03 创作工作室** - AI 内容生成演示

#### 多屏模式
1. 点击右上角的 **多屏图标** 按钮
2. 三个屏幕同时显示
3. 可以切换横向/纵向布局

## 🐛 故障排查

### 页面无法加载
1. 确认开发服务器已启动
2. 检查端口 3000 是否被占用
3. 查看控制台错误信息

### 动画卡顿
1. 关闭其他标签页
2. 检查系统资源
3. 更新浏览器
4. 降低动画复杂度

### 数据不显示
1. 检查浏览器控制台
2. 确认 JSON 数据文件存在
3. 验证数据格式正确

## 💡 提示

### 最佳体验
- 使用现代浏览器（Chrome/Edge/Firefox）
- 推荐屏幕分辨率 1920×1080 或更高
- 多屏模式建议使用大屏幕

### 快捷操作
- 单屏模式：快速切换查看不同功能
- 多屏模式：全局监控所有数据
- 横向布局：适合宽屏显示器
- 纵向布局：适合竖屏或多显示器
