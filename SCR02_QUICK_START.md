# 🚀 SCR-02 V2 快速开始指南

## 5分钟快速体验

### 步骤 1: 启动开发服务器

```bash
cd Claw3D
npm run dev
```

等待服务器启动，看到：
```
✓ Ready in 3.2s
○ Local:   http://localhost:3000
```

### 步骤 2: 访问应用

打开浏览器访问：
```
http://localhost:3000/office
```

### 步骤 3: 查看 SCR-02 V2

1. 页面加载后，默认显示 SCR-01（数字办公室）
2. 点击顶部的 **"SCR-02 舆情分析中心"** 标签
3. 看到新版本界面，右上角显示 **"V2 ✨"** 徽章

### 步骤 4: 探索功能

#### 左列
- **全平台对比** - 8个平台热度条形图
- **热力图** - 24小时×8类别热度展示

#### 中列
- **情感分析** - 正负面情绪分析
- **话题网络** - 交互式话题关联图

#### 右列
- **预警系统** - 风险预警和机会识别
- **人群画像** - 年龄、地域、活跃时段分析

### 步骤 5: 切换版本（可选）

点击版本徽章旁的 **"切换版本"** 按钮，可以切换到 V1 查看对比。

## 常见问题

### Q: 如何切换回旧版本？
A: 点击版本徽章旁的"切换版本"按钮即可。

### Q: 数据是真实的吗？
A: 目前使用 Mock 数据，可以在后续对接真实 API。

### Q: 如何自定义颜色？
A: 修改各组件的 CSS Module 文件中的颜色变量。

### Q: 支持移动端吗？
A: 支持响应式布局，但最佳体验在桌面端。
