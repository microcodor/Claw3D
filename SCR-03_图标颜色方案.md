## 颜色系统

### Tailwind CSS 颜色映射

```css
#3b82f6  → blue-500    (蓝色)
#ef4444  → red-500     (红色)
#10b981  → emerald-500 (绿色)
#f59e0b  → amber-500   (琥珀色)
#a855f7  → purple-500  (紫色)
#ff9500  → 自定义橙色   (主题色)
```

### 颜色语义

1. **蓝色 (#3b82f6)** - 文档、文章、信息
   - 代表专业、可信、稳定
   - 用于文章类型和任务数统计

2. **红色 (#ef4444)** - 视频、热度、警示
   - 代表活力、热度、重要
   - 用于视频类型和热度指标

3. **绿色 (#10b981)** - 社交、成功、产出
   - 代表成长、成功、互动
   - 用于社交类型和字数统计

4. **琥珀色 (#f59e0b)** - 广告、时间、提醒
   - 代表注意、时间、推广
   - 用于广告类型和耗时统计

5. **紫色 (#a855f7)** - 统计、分析、进度
   - 代表创新、分析、智能
   - 用于完成率统计

6. **橙色 (#ff9500)** - 主题色
   - 代表创意、灵感、能量
   - 用于组件标题图标

## 视觉效果

### 对比度
所有颜色都在深色背景上有良好的对比度：
- 最小对比度：4.5:1（符合WCAG AA标准）
- 推荐对比度：7:1（符合WCAG AAA标准）

### 色彩平衡
- 暖色调（红、橙、琥珀）：40%
- 冷色调（蓝、绿、紫）：60%
- 整体色彩平衡，不会过于刺眼

### 视觉层次
1. **主题色（橙色）** - 最高层级，用于标题
2. **类型色（蓝/红/绿/琥珀）** - 中等层级，用于分类
3. **统计色（多色）** - 数据层级，用于信息展示

## 可访问性

### 色盲友好
颜色选择考虑了色盲用户：
- ✅ 红绿色盲：使用不同的亮度和饱和度
- ✅ 蓝黄色盲：避免纯蓝和纯黄的组合
- ✅ 全色盲：保持足够的明度对比

### 高对比度模式
所有图标在高对比度模式下仍然清晰可见。

## 未来优化

### 主题切换
可以根据主题（亮色/暗色）动态调整图标颜色：
```tsx
const iconColor = theme === 'dark' ? '#3b82f6' : '#1e40af';
```

### 动画效果
可以添加颜色过渡动画：
```tsx
style={{ 
  color: '#3b82f6',
  transition: 'color 0.3s ease'
}}
```

### 悬停效果
可以添加悬停时的颜色变化：
```css
.icon:hover {
  color: #60a5fa; /* 更亮的蓝色 */
}
```
