## 🚀 快速开始

### 本地开发
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问应用
http://localhost:3000/office
```

### 服务器部署
```bash
# 使用部署脚本
./server-deploy.sh

# 或手动部署
npm run build
pm2 start ecosystem.config.js
```

## 📌 注意事项

1. **OpenClaw 依赖**: 本项目是 OpenClaw 的前端，需要 OpenClaw Gateway 运行才能显示完整功能
2. **环境配置**: 复制 `.env.example` 到 `.env` 并配置相关变量
3. **端口配置**: 默认端口 3000，可通过 `.env` 修改
4. **网络访问**: 如需局域网访问，设置 `HOST=0.0.0.0`
